#Descripción General
Este portafolio contiene una recopilación de los proyectos prácticos e integradores desarrollados a lo largo del semestre en la materia de **Programación Orientada a Objetos**. El objetivo principal es demostrar la correcta aplicación de paradigmas de desarrollo estructurado, la gestión de la persistencia de datos y la organización técnica del código fuente utilizando Git y GitHub como plataformas de control de versiones y documentación profesional.
A través de estos proyectos se evidencia el desarrollo de competencias clave en el diseño de software, tales como el análisis de problemas del mundo real, la abstracción de clases, la reutilización de código mediante relaciones de herencia y la manipulación segura de flujos de archivos.
